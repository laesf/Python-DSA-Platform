# Algorithms and Data Structures

An interactive software platform in Python for the study, implementation, and empirical measurement of the main algorithms and data structures. The project includes a graphical user interface (GUI) and a section on temporal complexity profiling.

## Key Features

The project is divided into independent modules, each of which is specifically designed to address a particular category of computational problems:

* **Sorting:** Implementation of Quick Sort, Merge Sort, Insertion Sort, Bubble Sort, Counting Sort, etc.
* **Linear Data Structures:** Singly Linked Lists, etc.
* **Search Trees:** 
  * Binary Search Tree (BST), etc.
  * Red-Black Tree (RBT) with automatic balancing, etc.
* **Hash Tables:** Collision via Chaining, Linear Inspection, Double Hashing, etc.
* **Graphs:** Representations using Adjacency Lists (directed and undirected, weighted and unweighted), etc. Visiting algorithms, Topological Sorting, Minimum Spanning Tree (Kruskal), Shortest Paths (Dijkstra), etc.
* **Performance & Profiling:** Modules for calculating execution times

## Prerequisites and Installation

To run the project, you must have **Python 3.x** installed on your system. The graphical interface uses `tkinter` (which is often included in the standard Python installation).

For performance graphs, you need to install the `matplotlib` library.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com//laesf/Python-DSA-Platform.git](https://github.com//laesf/Python-DSA-Platform.git)
    cd Python-DSA-Platform
    ```

2. **Install dependencies:**
    ```bash
    pip install matplotlib
    ```
